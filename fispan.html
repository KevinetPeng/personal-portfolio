Developing a Fast Changing Product While Writing Enterprise Grade Code

My internship at FISPAN: Sept 2021 - Dec 2021

During my internship at FISPAN, an early-stage Fintech startup, I developed a product full-stack and learned to write robust, maintainable code while also rapidly shipping new features for products directing over 75k total monthly transactions. I wore multiple hats and worked with many parts of the codebase, from publishing reusable frontend components, implementing backend-for-frontend API layers, to UI design, etc.

Outcomes

• Developed multiple micro-frontends in addition to publishing reusable features to internal NPM packages.
• Implemented backend-for-frontend API layers, and solved a build tool bug, reducing built times for myself and other engineers.
• Worked with product managers and UI/UX designers to transform user needs and designs into functioning products, in addition to developing my own UI designs on the fly for missing components.

Tech Stack

Node.js, React.js, Typescript, MongoDB, Redux
